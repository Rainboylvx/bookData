# Rbook

## 简介

**Rbook**是一本~~开源~~的算法书，是`Rainboy`写给自己学生看的书,使用`C++`语言描述,它由以前的两本书(《CppBook 入门书》,《NoipBook 算法书》)合并而来.目前由作者不断的完善中!
