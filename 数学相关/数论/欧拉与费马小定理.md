# 欧拉与费马小定理

## 欧拉定理

在数论中，欧拉定理,（也称费马-欧拉定理）是一个关于同余的性质。欧拉定理表明，若n,a为正整数，且n,a互质，则:

```math
a^{\varphi(n)} \equiv 1(\bmod\ n)
```

其中$$\varphi(n)$$表示$$[1,n]$$有几正整数与$$n$$互质

## 费马小定理

对于素数$$p$$,$$\varphi(p)=p-1$$

若正整数$$a$$与素数$$p$$互质，则有:

```math
a^{p-1} \equiv 1(\bmod\ p)
```

证明这个定理非常简单，由于 φ(p) = p -1，代入欧拉定理即可证明。
