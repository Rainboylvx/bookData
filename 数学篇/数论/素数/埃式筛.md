## 埃式筛

<!-- template start -->
```c
const int maxn = 1e5+5;
bool notp[maxn]; // false 表示素数,true非素数
void aPrime(){
    memset(notp,false,sizeof(notp));
    notp[0] = notp[1] = true;
    int i,j;
    for(i=2;i<=maxn;i++)
        for( !notp[i]){
            //防止i*i 溢出,或用long long 替代
            if( i > maxn/i) continue;
            //直接从i*i开始就可以了,小于i倍的已经被筛选过了
            for(j=i*i;j<=maxn;j+=i)
                notp[j] = true;
        }
}
```
